Master Tables (Reference Data)

| Table                       | Fields                                                                                                                                                                                                      | Notes                                                    |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| **master_roles**            | `id PK, role_name VARCHAR(50) UNIQUE, description VARCHAR(255), created_at DATETIME, updated_at DATETIME, is_active BOOLEAN`                                                                                | Default roles: Admin, Sales, Technician, Accounts, Store |
| **master_users**            | `id PK, name VARCHAR(100), email VARCHAR(150) UNIQUE, password_hash VARCHAR(255), role_id FK(master_roles.id), created_at, updated_at, is_active`                                                           | Employee or system user                                  |
| **master_leads**            | `id PK, name VARCHAR(100), email VARCHAR(150), phone VARCHAR(20), source VARCHAR(50), status VARCHAR(50), created_at, updated_at, is_active`                                                                | Status: New, Contacted, Demo Scheduled                   |
| **lead_demo_schedule**      | `id PK, lead_id FK(master_leads.id), scheduled_date DATETIME, employee_id FK(master_employees.id), feedback TEXT, created_at, updated_at, is_active`                                                        | Demo scheduling & feedback                               |
| **master_clients**          | `id PK, name VARCHAR(100), gst_number VARCHAR(50), contact_name VARCHAR(100), contact_email VARCHAR(150), contact_phone VARCHAR(20), created_at, updated_at, is_active`                                     | Converted leads become clients                           |
| **master_client_locations** | `id PK, client_id FK(master_clients.id), address TEXT, city VARCHAR(50), state VARCHAR(50), pincode VARCHAR(10), contact_person VARCHAR(100), contact_phone VARCHAR(20), created_at, updated_at, is_active` | Client can have multiple locations                       |
| **master_employees**        | `id PK, name VARCHAR(100), role_id FK(master_roles.id), mobile VARCHAR(20), email VARCHAR(150), created_at, updated_at, is_active`                                                                          | Employees assigned to tasks, demos, machines             |
| **master_machines**         | `id PK, machine_name VARCHAR(50), serial_number VARCHAR(50), model VARCHAR(50), created_at, updated_at, is_active`                                                                                          | Machine master                                           |
| **machine_installations**   | `id PK, machine_id FK(master_machines.id), client_location_id FK(master_client_locations.id), installed_on DATETIME, otp VARCHAR(6), confirmed BOOLEAN DEFAULT FALSE, created_at, updated_at, is_active`    | Installation & OTP confirmation                          |
| **master_items**            | `id PK, item_name VARCHAR(50), unit VARCHAR(20), created_at, updated_at, is_active`                                                                                                                         | Raw materials: Tea, Milk, Sugar, Cups                    |
| **item_stock**              | `id PK, item_id FK(master_items.id), quantity FLOAT, last_updated DATETIME, created_at, updated_at, is_active`                                                                                              | Current stock per item                                   |
| **master_suppliers**        | `id PK, name VARCHAR(100), email VARCHAR(150), phone VARCHAR(20), address TEXT, created_at, updated_at, is_active`                                                                                          | Suppliers of raw materials & machine parts               |


Transaction Tables (Business Operations)
| Table                    | Fields                                                                                                                                                                                                                              | Notes                         |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------- |
| **purchase_orders**      | `id PK, supplier_id FK(master_suppliers.id), order_number VARCHAR(50) UNIQUE, total_amount FLOAT, created_at, updated_at, is_active`                                                                                                | Purchase orders raised        |
| **purchase_order_items** | `id PK, purchase_order_id FK(purchase_orders.id), item_id FK(master_items.id), quantity FLOAT, rate FLOAT, created_at, updated_at, is_active`                                                                                       | Items in each PO              |
| **goods_received**       | `id PK, purchase_order_id FK(purchase_orders.id), received_date DATETIME, received_by FK(master_employees.id), created_at, updated_at, is_active`                                                                                   | Goods received record         |
| **supplier_invoices**    | `id PK, purchase_order_id FK(purchase_orders.id), invoice_number VARCHAR(50), invoice_date DATETIME, amount FLOAT, created_at, updated_at, is_active`                                                                               | Invoice against PO            |
| **supplier_payments**    | `id PK, supplier_invoice_id FK(supplier_invoices.id), payment_date DATETIME, amount FLOAT, mode VARCHAR(20), created_at, updated_at, is_active`                                                                                     | Payment to supplier           |
| **machine_consumption**  | `id PK, machine_id FK(master_machines.id), consumption_date DATE, created_at, updated_at, is_active`                                                                                                                                | Daily consumption per machine |
| **consumption_items**    | `id PK, consumption_id FK(machine_consumption.id), item_id FK(master_items.id), quantity FLOAT, created_at, updated_at, is_active`                                                                                                  | Material used by machine      |
| **invoices**             | `id PK, client_id FK(master_clients.id), invoice_date DATE, total_amount FLOAT, gst_amount FLOAT, minimum_billing FLOAT, created_at, updated_at, is_active`                                                                         | Monthly billing invoices      |
| **invoice_items**        | `id PK, invoice_id FK(invoices.id), item_id FK(master_items.id), quantity FLOAT, rate FLOAT, amount FLOAT, created_at, updated_at, is_active`                                                                                       | Item-wise billing             |
| **gst_details**          | `id PK, invoice_id FK(invoices.id), gst_rate FLOAT, gst_amount FLOAT, created_at, updated_at, is_active`                                                                                                                            | GST breakdown                 |
| **client_payments**      | `id PK, client_id FK(master_clients.id), invoice_id FK(invoices.id), payment_date DATE, amount FLOAT, payment_mode VARCHAR(20), created_at, updated_at, is_active`                                                                  | Client payments               |
| **tds_records**          | `id PK, client_payment_id FK(client_payments.id), tds_amount FLOAT, created_at, updated_at, is_active`                                                                                                                              | TDS deductions                |
| **debit_notes**          | `id PK, client_id FK(master_clients.id), invoice_id FK(invoices.id), amount FLOAT, reason VARCHAR(255), created_at, updated_at, is_active`                                                                                          | Debit adjustments             |
| **service_tickets**      | `id PK, machine_id FK(master_machines.id), client_location_id FK(master_client_locations.id), employee_id FK(master_employees.id), issue_type VARCHAR(50), description TEXT, status VARCHAR(50), created_at, updated_at, is_active` | Machine complaints            |
| **ticket_parts**         | `id PK, ticket_id FK(service_tickets.id), item_id FK(master_items.id), quantity FLOAT, created_at, updated_at, is_active`                                                                                                           | Parts used in service         |
| **service_history**      | `id PK, ticket_id FK(service_tickets.id), action_taken TEXT, action_date DATETIME, created_at, updated_at, is_active`                                                                                                               | Service log                   |
